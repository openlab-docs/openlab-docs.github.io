RESTful API¶

IRIS Discovery Service의 RESTful API에 대해 설명합니다.

RESTful API Description Category
auth IRIS Discovery Service 의 API 를 사용하기 위해 ID, Password 를 이용하여 토큰을 발행하기 위한 API  
query IRIS Discovery Service 에 검색 명령어를 통해 분석 결과를 도출 하는 API
